To create a new webpage on the production server:

  1. Create a new file in gedit called "index.html"
  2. Add basic html format
  3. Save index.html file
  4. Create a new file in gedit called "main.css"
  5. Add css content
  6. Save main.css file
  7. git add, commit, and push index.html and main.css files
  8. open sftp and login
  9. cd public_html
  10. put index.html and put main.css
